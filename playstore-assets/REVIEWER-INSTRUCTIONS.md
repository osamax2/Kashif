## Troubleshooting

### If Login Fails
1. Check internet connection
2. Ensure credentials are entered correctly:
   - Email: reviewer@kashifroad.com
   - Password: ReviewTest2026!
3. Try alternative account if needed

### If Permissions Don't Work
1. Go to device Settings → Apps → Kashif
2. Enable Location, Camera, and Notifications permissions manually

### If Map Doesn't Load
1. Ensure location permission is granted
2. Check internet connection
3. Wait 10-15 seconds for map to initialize
